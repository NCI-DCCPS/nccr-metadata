NCCR Metadata

RDF vocabulary and DATMM metadata for the National Childhood Cancer Registry Data Platform

Ontology Files

FilePurposeFormat
nccr_vocab.ttl Vocabulary — classes and properties for variables, value sets, filters, processing rules TurtleOWL
nccr_datmm.ttl Dataset catalog — DATMM 6.0.0 records for the platform and its 9 datasources TurtleDATMM
nccr_instances.ttl Instance data — all 533 variables across 9 datasources with value sets, filters, and processing rules (28,620 triples) Turtle

Quick Start

Load into Python with rdflib:

from rdflib import Graph

g = Graph()
g.parse("https://nci-dccps.github.io/nccr-metadata/nccr_vocab.ttl")
g.parse("https://nci-dccps.github.io/nccr-metadata/nccr_datmm.ttl")
print(f"Loaded {len(g)} triples")
